## О проекте

- Сайт представляет собой платформу личной библиотеки. Где каждому пользователю показываются только его книги.

- Сделана регистрация, логин и логаут пользователя.

- Используются формы с полной валидацией.

- Пользователи могут добавлять книги, устанавливать прогресс чтения, ставить рейтинг и писать отзывы о книге. Так же пользователь может устанавливать или изменять "Цель на год" - это запланированное количество книг к прочтению.

- Прогресс "Цели на год" показывается на главной странице и в сайдбаре. Так же там же показываются прогресс чтения и последни прочитанные книги.

- Сделан поиск по названию или автору книги. Так же сделан фильтр, согласно статусам прочтения книг.

- Роутинг страниц защищен.

- Приложение общается с удаленной PostgreSQL базой данных Supabase. Используя механизмы "Супабейс", запросы к базе происходят на основе встроенных API.

- Приложение адаптировано.
